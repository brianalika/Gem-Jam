Gem Jammin'!

Directions:

At the start of the game, you will be given a random target number of gems to mine.

Below, there are four gems. By clicking on a gem, you will be awarded a specific number of points.

In order to win, you much match the number of gems you've collected with the target number. Be careful! The cave will collapse if you go above the target number, and you'll lose that round.

Each time the game restarts, a new value will be given to each gem.
